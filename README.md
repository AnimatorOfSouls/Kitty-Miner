# Kitty Miner
Discord bot about mining and trading cats.

## Cat Tiers
- White
- Blue
- Green
- Orange
- Red
- Pink
- Purple


## Commands
command (alternative names) - description

- newprofile - creates a new profile
- profile (p) - view profile
- tiers - view the tiers of cats

- mine (m) - mine a cat
- inventory (inv) - view inventory
- recipes - view recipes and their recipe IDs
- craft - craft an item, used in the format "cat craft recipeID" (e.g. cat craft blue)
- trade - trade cats with another player
